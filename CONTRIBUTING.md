# Katkıda Bulunma Rehberi

Bu projeye katkı sağladığınız için teşekkürler! Profesyonel bir çalışma ortamı için lütfen şu adımları izleyin:

1. Bu depoyu forklayın.
2. Yeni bir branch açın (`feat/ozellik-adi` veya `fix/hata-adi`).
3. Kodlarınızı proje standartlarına (Linting, Dosya yapısı) uygun şekilde yazın.
4. Anlamlı commit mesajları yazın (Conventional Commits).
5. Değişikliklerinizi içeren bir Pull Request (PR) açın.

Sorularınız için [Issue](https://github.com/devolopereph/projects/issues) açabilirsiniz.
